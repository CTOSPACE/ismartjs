确认框控件

调用方法

S.confirm(msg, sign|option).done(function(){

   //这里是点击确认按钮的时候调用的

}).fail(function(){
    //这里是点击取消按钮调用的
    });

参数

  1. msg 要显示的信息
  2. sign|option：标识或者option对象，sign的值为：info、 warning、success、 danger，分为不同的级别，默认为 warning

option 的格式为： {title : "提示", sureBtnName: "确定", cancelBtnName:"取消", sign: "info"}

示例

源码

<p><input s="click" s-click-action="confirmDialog()" type="button" class="btn btn-primary" value="点击弹出确认框"  /> </p>
<script type="text/javascript">
    function confirmDialog(){
         S.confirm("你确认删除所选吗？", "warning").done(function(){
                S.alert("你点了确认按钮");
            }).fail(function(){
                S.alert("你点了取消按钮");
            });
    }
</script>
