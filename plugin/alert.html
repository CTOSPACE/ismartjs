alert警告框控件

该控件是 Smart 原型(prototype)扩展控件，直接通过 S.alert(msg, sign, title) 进行调用。

调用方法

S.alert(msg[, sign[, title|option]]) 返回值：$.Deferred ，当警告框关闭的时候，会调用 deferred.resolve() 通知，可以通过 deferred.done(fn) 来监听警告框关闭。

参数说明

  1. msg：提示的文字
  2. sign：标识，值为：info、warning、success、danger，分为不同的级别。
  3. title|option：警告框的标题或者option对象

option 的格式为 {title:'...', btnName:'...'}， btnName是关闭按钮的名称， 默认为确定

示例


源码

<select class="form-control" id="alertSign" style="width: 300px;">
    <option value="info">info</option>
    <option value="warning">warning</option>
    <option value="success">success</option>
    <option value="danger">danger</option>
</select>
<input s="click" s-click-action="sAlert()" type="button" class="btn btn-primary btn-small" value="点击弹出警告框" />
<script type="text/javascript">
    var alertSignSelect = S.N("#alertSign");
    function sAlert(){
        S.alert("你点击的是警告框，点击下面关闭按钮关闭。", alertSignSelect.val(), "警告").done(function(){
            S.alert("你关闭了上一个警告框。");
        });
    }
</script>
