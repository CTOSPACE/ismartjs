dataSubmit

该控件用于自动提交相关数据到远程，可以和 loop、resource、check、cdsc 控件组合起来实现如：自动加载文章列表，选择相应的文章，删除所选的文章。该过程不需要书写任何JS代码，只需要做相关配置即可

1. 配置参数

  1. s：dataSubmit
  2. ctx:data：要提交的数据，该参数是上下文参数， 如果该参数返回方法，则把方法的执行结果做为数据提交
  3. url：提交到远程的url
  4. type：提交的方式[get,post,remove,put]，默认为post
  5. ctx:listener：监听器，监听器格式为： 
    {
    onDone: function(){},//提交成功,
    onFail: function(){}//提交失败
}

2. 事件

  1. submit-done：当提交数据成功的时候，会触发该事件，参数为远程的返回值
  2. submit-fail：当提交数据失败的时候，会触发该事件
