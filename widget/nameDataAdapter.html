nda 将所有拥有 name 属性的元素适配成基础控件

该控件是将其node的子元素中拥有 name 属性的元素适配成基础控件，如果已经是控件，则不作处理， 如果子元素没有 s-smart-alipayKey 属性，就将 name 属性的值设置为 s-smart-alipayKey属性的值。

控件定义 配置参数

  1. s：nda

示例

<form s="nda">
    <input type="text" name="email" />
    <input type="text" name="password" />
    <input type="text" name="address" s-smart-alipayKey="adr" />
</form>
上述form会被适配成如下：
<form s="nda">
    <input w="" s-smart-alipayKey="email"  type="text" name="email" />
    <input w="" s-smart-alipayKey="password"  type="text" name="password" />
    <input w="" type="text" name="address" s-smart-alipayKey="adr" /> <!--该元素拥有 s-smart-alipayKey 属性，则处理 s-smart-alipayKey 的属性值。-->
</form>
