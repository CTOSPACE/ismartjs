浏览器事件控件

该类控件用于封装浏览器的鼠标响应事件，以使得事件主体可以在当前页面窗口的闭包中运行。

1. event

参数配置

  1. s：event
  2. type：指定监听的是浏览器的什么事件，如：click,change,focus等等所有浏览器的原生事件。
  3. action：事件执行的主体代码。

示例：注册按钮的点击事件

源码

<input type="button" s="event" s-event-type="click" s-event-action="eventOfClick()" value="点击我！" />
<script type="text/javascript">
    function eventOfClick(){
        alert("你点击我干嘛！！！！");
    }
</script>

禁用操作按钮

通常在提交表单的时候，需要在提交动作的开始时禁用按钮，在提交成功之后再使按钮可用，这样可以确保重复提交的问题。 event 控件提供了简易的方法实现这样的功能。示例如下：

<input s="click" s-click-action="return waitOperate()" type="button" value="点击我，3秒之后解除禁用状态" class="btn btn-primary"/>
<script type="text/javascript">
    function waitOperate(){
        var deferred = $.Deferred();
        setTimeout(function(){
            deferred.resolve();//3秒后通知完成，同样适用 deferred.reject()
        }, 3000);
        return deferred;//这里需要返回deferred才能自动设置按钮状态，同样在 s-click-action 的配置中也需要返回。
        //如果是ajax的操作，直接返回ajax操作的返回值，在ajax执行完成后会自动解除按钮禁用状态
        //return S.post(...);
    }
</script>
