控件的声明

通过定义元素的 s 属性值来定义元素的控件，如： s="resource,loop"，多个声明用逗号隔开

控件参数的声明

为了保证控件的参数属性唯一，参数对应的属性配置都有一个namespace，这个namespace由控件的id和参数名构成， 如：resource 控件的 data-res 参数对应的属性名为 s-resource-data-res

示例：

 <table s="resource,loop" s-resource-data-res="json/users.json" s-loop-type="tree">

tpl 模板控件 <div s="tpl" ></div> 详细描述 api
resource 资源加载控件，通过该控件可以自动加载服务器的资源 <div s="resource" s-resource-data-res="xxx.json" ></div> 详细描述 api
datac 为子元素赋值控件 <div s="datac" ></div> 详细描述
nda 全称为name data adapter,将该控件的子元素中所有拥有name属性的元素适配成的Smart控件。 
<div s="nda"  >
    <input type="text" name="aaa" />
    <span name="bbb" ></span>
</div>
详细描述
loop 循环控件，通过该控件可以构建 列表，表格（grid） s='loop' 详细描述
check 选择控件，可以为列表、表格添加行选中功能 s='check' 详细描述
dataSubmit 该控件可以和 loop、resource、check、cds 控件组合起来实现如：自动加载文章列表，选择相应的文章，删除所选的文章。该过程不需要书写任何JS代码，只需要做相关配置即可 s='dataSubmit' 详细描述
cds 全称为：check data submit，该控件可以和 loop、resource、check、dataSubmit 控件组合起来实现如：自动加载文章列表，选择相应的文章，删除所选的文章。该过程不需要书写任何JS代码，只需要做相关配置即可 s='check' 详细描述
