loop控件

可用此控件构筑 列表，表格。

1. 控件定义

s="loop"

参数配置

  1. type：tree|为空 表示循环的类型，如果为tree，则表示进行树循环，默认为空
  2. tree-c：默认值children 当进行树循环时，子元素的key。
  3. tree-indent-width：默认值20 当进行树循环时，子元素的缩进值，默认为20px。
  4. tree-indent-str：默认值为空 当进行树循环时，子元素的缩进字符，当该值不为空时，tree-indent-width属性则无效

其他配置

  1. s-loop-tree-role='indent'：当以tree循环，并且需要缩进的时候，需要为缩进的元素配置该属性。

2. 示例

姓名 体积 类型

源码

<table class="table table-bordered table-striped table-hover" style="width: 500px;">
    <thead>
    <tr>
        <th>姓名</th>
        <th>体积</th>
        <th>类型</th>
    </tr>
    </thead>
    <tbody s="loop" s-smart-data="ctx:userData" >
    <tr s-loop-role="row" s="datac,row">
        <td s s-smart-key="name"></td>
        <td s s-smart-key="tj"></td>
        <td s s-smart-key="kind"></td>
    </tr>
    </tbody>
</table>
<script type="text/javascript">
    var userData = [
        {
            name: "Jerry",
            tj: "Small",
            kind: "Mouse"
        },
        {
            name: "Tom",
            tj: "Big",
            kind: "Cat"
        }
    ];
</script>

ctx:userData 表明该值是闭包中的 userData 的值。
