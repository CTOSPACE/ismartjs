submit按钮控件

监听表单的回车提交事件，并将此事件作用于该按钮的click事件。

参数配置

  1. s：submit
  2. ctx:action：提交表单要执行的方法， 该参数是控件元素属性s-submit-action的值
  3. ctx:done：提交成功时执行的方法，该方法接受一个参数，即提交结果的返回值。
  4. ctx:fail：提交失败时执行的方法
  5. ctx:always：提交完成时执行的方法，不管是执行成功还是失败了。
  6. reset：是否在提交成功时reset form，默认为false。

根据原始表单配置提交表单

亦可以根据表单的原始配置参数：action,method,enctype去自动提交表单，该提交动作会先验证表单，再去提交，提交后会发出2个事件：

事件

  1. submit-done：提交成功事件，事件参数为远程服务器的返回值
  2. submit-fail：提交失败，事件参数为xhr对象

示例1 根据表单原始属性去提交表单

源码

<form class="form-inline" id="form1"
      s="submit" s-submit-done="submitDone" action="json/submit.json" method="get">
    <div class="form-group">
        <label class="sr-only" >Name</label>
        <input type="text" name="name" class="form-control" placeholder="Enter name" />
    </div>
    <div class="form-group">
        <label class="sr-only" >Phone</label>
        <input type="text" name="phone" class="form-control" placeholder="Phone" />
    </div>
    <input s="click" type="submit" class="btn btn-primary " value="查询" />
</form>
<script type="text/javascript">
    function submitDone(rs){
        S.alert("提交成功，返回值为：" + JSON.stringify(rs));
    }
</script>
